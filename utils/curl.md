# curl
	-X: POST|DELETE|GET: Define el metodo HTTP
	-d: Envia datos en el cuerpo del request
	-H "Header-Name: something": Agrega headers personalizados 
	-i: Muestra los headers de respuesta
	-s: Elimina la salida de progreso
	-v: Verbose
	-L: Sigue redirecciones automaticamente
	-o <archivo>: Guarda la respuesta en un archivo
	-u user:pass: Autenticacion basica HTTP
	--compressed: Solicita una respuesta comprimida
	--socks5: Permite usar un proxy SOCKS5 para enrutar la conexion
