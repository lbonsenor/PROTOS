# dig
	+short: Muestra la salida "cruda"
	+noall +answer: Muestra solo la seccion de respuesta
	@servidor: Especificar un servidor DNS: @localhost|@8.8.8.8
	ANY: Solicita todos los registros disponibles
	A: Consulta el registro de direccion IPv4
	AAAA: Consulta el registro de direccion IPv6
	MX: Consulta los servidores de correo
	NS: Muestra los servidores de nombres para un dominio
	CNAME: Muestra los alias
	TXT: Muestra registros de texto
	SOA: Muestra informacion sobre la zona DNS
	+trace: Hace una resolucion paso a paso
	+dnssec: Solicita informacion de seguridad en DNS
	+x IP: Realiza una resolucion inversa
	-t: Especificar el tipo de registro DNS que quiere consultar
