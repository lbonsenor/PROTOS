1. `sudo apt install openssh-server`
2. `systemctl status ssh`
3. En /etc/ssh/sshd_config: `PermitRootLogin no`
4. Si hacemos `ssh localhost`: Nos pide la password de la VM

Para autentificacion por claves:
1. En /etc/ssh/sshd_config: `PubkeyAuthentication yes`
2. Generamos el par de claves con `ssh-keygen`
3. Nos autenticamos con el servidor con la clave publica: `ssh-copy-id localhost`

Warning: EN ~/.ssh se guardan archivos del servidor y del cliente

4. Averiguamos el fingerprint: `ssh-keygen -l -f /etc/ssh/ssh_host_rsa_key.pub`

Para conectarse utilizando ssh foo:
1. Averiguo la ip: `ifconfig`
2. Desde otra maquina, en /etc/hosts: `[IP] foo`
3. Nos podemos conectar con: `ssh lbonsenor@foo`

Ej: Pampero
1. `dig pampero.itba.edu.ar`
2. En /etc/hosts: `[IP] foo`
3. `ssh lbonsenor@foo`
